# 🏦 ATM Based Bank Management System

---

## 📌 Overview

This project is an **ATM-based Bank Management System** that simulates real-world banking operations. It allows users to perform essential transactions like withdrawing money, depositing funds, checking balance, and fast cash through a simple and user-friendly interface.

## 🎯 Objective

The objective of this project is to design and develop a banking system simulation that replicates ATM functionalities. It demonstrates how digital banking transactions work while integrating frontend design, backend logic, and database management.

---

## ✨ Features

* ✅ User Registration (Sign Up)
* ✅ Secure Login System
* ✅ Balance Checking
* ✅ Cash Withdrawal
* ✅ Deposit Money
* ✅ Fast Cash Option
* ✅ ATM Interface Simulation
